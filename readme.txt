# README - Template Landing Page "Sambal Maknyus"

## Deskripsi
Template landing page modern untuk produk makanan (sambal) dengan:
✅ Desain responsif (mobile & desktop) 
✅ Form pemesanan + metode pembayaran
✅ Testimoni carousel
✅ Countdown promo otomatis
✅ Floating WhatsApp button

## File Structure
📁 Template_landingpage_sambal/
├── index.html          # File utama
├── images/             # Folder asset gambar
│   ├── img.png         # Foto produk utama
│   ├── merah.png       # Sambal bawang  
│   ├── hijau.png       # Sambal ijo
│   ├── tomat.png       # Sambal tomat
│   ├── bca.png         # Logo bank
│   └── ...             # DLL
└── README.txt          # File ini

## Cara Customisasi
1. GANTI GAMBAR:
   - Upload gambar produk baru ke folder /images
   - Update nama file di HTML (line 110, 116, 122)

2. UBAH TEKS:
   - Buka index.html dengan text editor
   - Cari dan ganti teks sesuai kebutuhan:
     • Nama produk (line 103)
     • Harga (line 189-223)  
     • Testimoni (line 327-414)

3. UPDATE KONTAK:
   - Nomor WhatsApp (ganti di line 85 dan 284)
   - Rekening bank (line 543-563)
   - Alamat (line 643)

4. WARNA:
   - Edit class Tailwind di HTML (utama: red-400, purple-100)

## Teknologi
• Tailwind CSS (via CDN)
• AOS Animation Library  
• Vanilla JavaScript
